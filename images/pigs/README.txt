Put your favorite pig photos in this folder (jpg, png, or webp).

Then open about.html, find the `pigPhotos` array near the bottom of the
page (search for "pigPhotos"), and add a line for each photo:

  { src: "images/pigs/my-pig-1.jpg", caption: "the cutest one I've seen" }

Caption is optional. These show up in a popup when someone clicks the
word "pigs" in your About page text.
